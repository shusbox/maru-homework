| **Type**     | **설명**                 | **참고사항**                             |
| ------------ | ------------------------ | ---------------------------------------- |
| **feat**     | 기능 추가                |                                          |
| **build**    | 라이브러리 세팅/업데이트 |                                          |
| **config**   | Config 파일 설정         |                                          |
| **remove**   | 파일 또는 폴더 삭제      |                                          |
| **rename**   | 파일 또는 폴더 수정      |                                          |
| **fix**      | 기능 에러 수정           |                                          |
| **style**    | UI 관련 CSS 수정         |                                          |
| **chore**    | 단순 코드 수정           | 주석, 줄 바꿈, 임포트문, 세미콜론 등등.. |
| **refactor** | 기능 변경 없는 코드 개선 |                                          |
| **asset**    | 이미지, 폰트             | 리소스 파일                              |
| **docs**     | 문서 작성                |                                          |
